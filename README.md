## 📌 Project Overview

**Forensic Data Analytics for Anomaly Detection in Evolving Networks Using Wireshark**

This project focuses on analyzing network traffic to detect suspicious activities and anomalies in modern, dynamic network environments. With the rapid growth of cloud computing, IoT devices, and continuously changing network structures, traditional monitoring methods are often insufficient. This system leverages forensic data analytics to provide deeper insights into network behavior and enhance security monitoring.

---

## 🧠 Concept & Idea

In today’s networks, data flows continuously like a busy highway. Among thousands of normal packets, a few malicious ones may quietly attempt to breach the system. This project acts as an intelligent monitoring system that captures, inspects, and analyzes these packets to identify abnormal patterns that could signal cyber threats.

---

## ⚙️ Working Methodology

The system operates in multiple stages:

### 1. Packet Capture

Network traffic is captured in real time using Wireshark, which records detailed packet-level data from the network.

### 2. Data Extraction

Important attributes are extracted, including:

* Source and Destination IP addresses
* Protocol types (TCP, UDP, HTTP, etc.)
* Packet size and frequency
* Time-based information (timestamps)

### 3. Data Analysis

The extracted data is processed to understand normal network behavior. Statistical or rule-based methods are used to identify deviations.

### 4. Anomaly Detection

Any unusual activity such as:

* Sudden spikes in traffic
* Repeated requests from a single IP
* Unknown or suspicious connections
  is flagged as a potential anomaly.

### 5. Reporting

The system generates logs or reports that help in identifying and investigating possible security threats.
